# Epic List

## Proposed Epics

1. **Epic 1: Foundation & Infrastructure** (6 stories)

   - Establish .NET 8 backend project, PostgreSQL database, JWT authentication, basic API structure, and AWS deployment foundation

2. **Epic 2: Contractor Management & Scoring Engine** (6 stories)

   - Implement contractor CRUD operations, availability calculation, distance/proximity integration with mapping API, and the intelligent scoring & ranking algorithm (the core differentiator)

3. **Epic 3: Dispatcher Portal & Job Assignment** (6 stories)

   - Build dispatcher dashboard UI, job list view, contractor recommendations display, one-click assignment workflow, and contractor list management (add/remove/filter)

4. **Epic 4: Job & Customer Portal** (6 stories)

   - Implement customer job submission form, real-time job status tracking view, assigned contractor visibility (ratings/reviews), and customer feedback/rating form

5. **Epic 5: Contractor Portal & Real-Time Notifications** (6 stories)

   - Build contractor job list view, job details modal, accept/decline workflow, job completion tracking, and SignalR-powered real-time notifications across all three roles

6. **Epic 6: Reviews, Email & System Coordination** (6 stories)

   - Implement customer rating system, average rating aggregation, email notification service (AWS SES), event publishing infrastructure, and end-to-end event coordination between all three portals

7. **Epic 7: Testing, Performance & Production Deployment** (7 stories)
   - Write integration test suite covering complete workflows, performance optimization (caching, database indexing), error handling polish, AWS deployment automation (CI/CD), and documentation
